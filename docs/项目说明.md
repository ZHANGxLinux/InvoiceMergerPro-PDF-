# 发票合并助手 — 项目说明

## 一、软件简介

**发票合并助手** 是一款面向财务场景的桌面工具，用于将多张发票 PDF 或图片素材，按可配置的网格规则排版到 A4 页面中，最终合并导出为一个 PDF 文件。

本工具采用 **PySide6** 构建界面，**PyMuPDF** 处理 PDF，**Pillow** 处理图片转换，支持 Windows / macOS / Linux。

## 三、核心模块说明

### 3.1 LayoutEngine（排版引擎）

负责以下工作：

1. **素材标准化**：PDF 提取首页；图片转单页 PDF
2. **网格排版**：按行数 × 列数将素材填入 A4 页面，等比居中
3. **合并输出**：生成完整 PyMuPDF 文档对象并保存
4. **预览渲染**：将所有页面渲染为 PNG 长图供界面显示

关键方法：

- `build_merged_document()` — 合并导出
- `render_preview_bytes()` — 生成预览图
- `release_scratch_files()` — 清理临时文件

### 3.2 WorkbenchWindow（主界面）

三栏布局：

- **左侧**：文件队列 `DocumentQueue`
- **中间**：排版参数（方向、行列滑块）与导出按钮
- **右侧**：滚动预览区 `ScrollPreviewPane`

所有耗时操作通过 `PreviewRenderJob` / `ExportRenderJob` 在后台线程执行，避免界面卡顿。

## 四、使用流程

1. 启动：`python run.py`
2. 点击 **添加文件**，选择 PDF 或图片（可多选）
3. 在 **排版设置** 中调整页面方向、行数、列数
4. 右侧 **实时预览** 自动刷新，可滚动查看全部页面
5. 确认无误后点击 **PDF 合并并导出**，选择保存路径

---

## 五、排版规则

| 项目 | 说明 |
|------|------|
| 页面尺寸 | A4 |
| 页面方向 | 纵向 / 横向 |
| 每页容量 | 行数 × 列数（默认 3×2 = 6 张） |
| 排列顺序 | 按文件列表顺序依次填入 |
| 缩放方式 | 等比缩放，居中，保留约 5% 边距 |
| PDF 素材 | 仅使用每个 PDF 的第一页 |
| 图片素材 | 自动转 PDF，透明底铺白 |

## 八、开发与打包

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行

```bash
python run.py
```

### Windows 打包

```bash
pip install pyinstaller
pyinstaller build.spec
```

生成的可执行文件位于 `dist/` 目录。

---

## 九、故障排查

- 若运行异常，请查看项目目录下的 **`runtime_error.log`**
- 预览生成较慢属正常现象（高清渲染需要更多计算）
- 大量文件合并时请耐心等待进度条完成
